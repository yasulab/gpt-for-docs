# gpt-for-docs
Proof of Concept: Can the generative AI automate tech contents translation?

## Sample files for PoC

These files below and their commit history are used as an experiment for ChatGPT to detect the diff in english and possible match up with japanese translation.

Sample files are fetched from [:octocat: yasslab/railsguides.jp](https://github.com/yasslab/railsguides.jp), which is licensed under the [CC BY-SA 4.0 International](https://github.com/yasslab/railsguides.jp#%E3%83%A9%E3%82%A4%E3%82%BB%E3%83%B3%E3%82%B9).

- JA: https://github.com/yasulab/gpt4docs/blob/main/guides/source/ja/classic_to_zeitwerk_howto.md
